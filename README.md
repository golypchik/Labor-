# Лабор.Картирование

Приложение для обработки данных картирования и создания отчетов по шаблонам.

## Установка

1. Установите Python 3.7 или выше
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Структура проекта

```
├── main.py                          # Главный файл приложения
├── requirements.txt                 # Зависимости
├── gui/                            # Модули графического интерфейса
│   ├── main_window.py              # Главное окно
│   ├── project_management_frame.py # Управление проектом
│   ├── key_elements_frame.py       # Ключевые элементы
│   ├── other_info_frame.py         # Прочая информация
│   └── tables_creation_frame.py    # Создание таблиц
├── data_processing/                # Обработка данных
│   └── excel_processor.py          # Обработка Excel файлов
├── report_generation/              # Генерация отчетов
│   └── report_generator.py         # Генератор отчетов
├── database/                       # Базы данных (создаются автоматически)
├── utils/                          # Вспомогательные модули
│   └── session_manager.py          # Менеджер сессии
├── image/                          # Изображения (логотип)
├── temp/                           # Шаблоны отчетов (.docx)
└── inform/                         # Временные файлы Excel (очищаются при закрытии)
```

## Использование

1. Запустите приложение:
```bash
python main.py
```

2. В разделе "Управление проектом":
   - Загрузите Excel файлы через кнопку "Подгрузить данные"
   - Выберите тип отчета (Объект хранения, Зона хранения, Холодильник/Морозильник)
   - Установите галочку "Учитывать влажность" при необходимости

3. Заполните данные в разделах:
   - "Прочая информация" - итоги картирования, области хранения, изображения, места рисков
   - "Создание таблиц" - диапазоны температур/влажности, периоды

4. Создайте отчет (кнопка "Создать отчет" - будет добавлена позже)

## Формат Excel файлов

Excel файлы должны содержать:
- Столбец A: общая информация (строка 5 - имя устройства/логгера)
- Столбец B: временные промежутки (начиная со строки 2)
- Столбец C: температурные значения (начиная со строки 2)
- Столбец D: значения влажности (начиная со строки 2)

## Примечания

- При закрытии приложения или завершении сессии все данные удаляются
- Файлы из папки `inform` автоматически удаляются при закрытии
- Логотип компании должен быть размещен в папке `image` с именем `logo.png`
- Шаблоны отчетов должны быть размещены в папке `temp`
